# 🕵️ OSINT Username Tracker

A Python open-source intelligence tool that searches for a username across 35+ platforms to map a target's digital footprint — built for ethical security research and cybersecurity learning.

---

## 📌 Features

- ✅ **Searches 35+ platforms** simultaneously
- ✅ **Multi-threaded** — fast parallel scanning
- ✅ **Digital footprint score** — exposure analysis
- ✅ **Compare two usernames** — find common platforms
- ✅ **Save results** to a text file
- ✅ **Progress bar** — real-time scan display
- ✅ **OSINT education mode** — learn the concepts
- ✅ **Privacy tips** — protect your own identity
- ✅ **No external libraries** — Python built-ins only
